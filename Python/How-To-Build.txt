 open cmd & run this

1. Install python : https://www.python.org/downloads/
check with this command after install : python --version 

2. python -m pip install --upgrade pip

3. pip install yt-dlp or python -m pip install yt-dlp

4. Download ffmpeg : https://www.gyan.dev/ffmpeg/builds/ 
Download the essentials files, open bin folder, and copy ffmpeg.exe & ffprobe.exe to this folder

How to install : https://youtu.be/KBnyOH1o5Ms?si=hySG-ngajx22b8wM
check after install : ffmpeg -version

or just run "download_ffmpeg.bat"

5. Run "-[build onedir]-.bat", after finish .exe will added in "Build" folder.
